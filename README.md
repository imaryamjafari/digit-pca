# Dimensionality Reduction with PCA on Handwritten Digits

Which numbers do you think are hardest for a computer to recognize? 

This project uses PCA to visualize how 8x8 pixel images of handwritten digits (0–9) are clustered. 
It shows which digits are easily separated, and which ones overlap and get confused.

---

## What's inside?

- Dimensionality reduction from 64 pixels to 2D / 3D using PCA
- Interactive scatter plots

---

## Tools and Libraries

- Python
- Numpy, Pandas, Scikit-Learn
- Plotly, Matplotlib
- Jupyter notebook
